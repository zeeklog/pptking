## 📦 依赖管理

### 核心依赖 (必需)
- React 19 + Next.js 15
- Zustand (状态管理)
- Radix UI (组件库)
- Tailwind CSS (样式)

### 可选依赖 (增强功能)
- pptxgenjs (PPTX导出)
- html-to-image (图片导出)
- jspdf (PDF导出)
- jszip (批量打包)
- echarts (图表渲染)
- katex (公式渲染)

### 安装方式
```bash
# 安装所有可选依赖
./install-optional-deps.sh

# 或按需安装
npm install pptxgenjs html-to-image jspdf jszip echarts katex
```
