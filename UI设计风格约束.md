# AI生成PPT系统 - 全新紫色系色彩系统设计规范
基于“专业、正式、富含AI风格”的核心需求，重新设计以**紫色系**为核心的色彩系统，通过科学的色相递进、明度/饱和度梯度，实现过渡自然、层次清晰的视觉效果，同时融入AI科技感元素（如渐变光效、低透明度科技纹理），适配PPT生成场景的专业性与易用性。


## 一、色彩系统核心设计理念
1. **紫色系为主轴**：选择波长适中的**蓝紫色调**（而非高饱和粉紫），既保留正式感，又避免视觉疲劳，符合办公场景使用需求。
2. **梯度递进原则**：主色、辅助色、点缀色按“明度递减、饱和度递增”规律排列，确保相邻色过渡自然（色相差≤30°，明度差≤20%）。
3. **AI科技感融合**：在纯色基础上叠加低透明度（5%-10%）的线性/径向渐变，模拟AI界面的“光感科技纹理”，避免传统紫色的厚重感。
4. **功能与色彩绑定**：核心操作（生成PPT、模板选择）用高识别度主色，辅助功能（编辑、导出）用次级色，状态提示（成功/加载）用对比色，符合用户操作预期。
5、系统必须支持**深色模式** 和 **浅色模式**；

## 四、渐变系统（AI科技感核心）
渐变是体现AI风格的关键元素，通过“主紫+辅助色”的线性/径向渐变，模拟“光效流动”效果，用于按钮、卡片、头部Banner等核心区域，避免单一纯色的沉闷。

### 1. 核心渐变（主紫色系）

## 六、阴影系统（层次与科技感）
阴影避免传统“纯黑阴影”，采用“主紫+低透明度”的阴影色，与主色调呼应，同时通过阴影模糊度区分界面层级（层级越高，阴影越柔和）。
